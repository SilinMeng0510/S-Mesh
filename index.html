" The MNIST database (Modified National Institute of Standards and Technology database) is a large database of handwritten digits that is commonly used for training various image processing systems. My project includes two parts. The first part is with parameters tunning and regular MLP implementation with a whole report. The second part is with implementation of CNN into the model. At least 99% of accuracy can be achieved with CNN and around 98% accuracy without CNN"


Titanic Binary Classification


" The Titanic database is a database of Titanic passengers' information that is recorded back then. We will be using the personal information of each passenger to train our model in order to predict if someone will survive if he/she is on the Titanic ship. This project is very easy, and I'm using a regular Multi-layers network to tackle this classification problem. In the end, 80% of accuracy can be achieved.
